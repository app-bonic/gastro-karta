# Gastro Karta

Interaktivna karta svih recenzija [Gastro Tražilice](https://gastrotrazilica.com/) — 180 restorana kroz 8 zemalja (Hrvatska, BiH, Srbija, Slovenija, Crna Gora, Makedonija, Austrija, Švicarska).

- filter po zemljama i pretraga
- Google ocjena, broj recenzija i adresa u popupu
- "Otvori recenziju" → originalni članak, "Uputi me" → Google Maps navigacija
- "Di sam ja?" → tvoja lokacija na karti
- svijetla podloga odzumirano, detaljna OSM zumirano

Statična stranica (Leaflet + OpenStreetMap/CARTO). Podaci i fotografije: gastrotrazilica.com — sva prava pripadaju njima; ovo je neslužbeni prikaz njihovih javnih recenzija.
